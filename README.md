## Megalo

**Megalo** 是基于 Vue（`Vue@2.5.16`） 的小程序开发框架，让开发者可以用 Vue 的开发方式开发小程序应用。**Megalo** 是为了跨 H5 和小程序两端的应用提供一个高效的解决方案，只需要少量改动即可完成 H5 和小程序之间的代码迁移。

**Megalo** 目前支持**微信小程序**，**支付宝小程序**，未来的版本将支持百度小程序。
